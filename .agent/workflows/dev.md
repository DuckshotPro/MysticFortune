# Start Local Development Server

This workflow starts the MysticFortune development server with all necessary setup.

// turbo-all

## Steps

1. Install dependencies (if needed)
```bash
npm install
```

2. Start the development server
```bash
npm run dev
```

The server will start on the configured port (usually http://localhost:5000).
