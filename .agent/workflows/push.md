# Quick Git Push

This workflow commits all changes and pushes to GitHub.

// turbo-all

## Steps

1. Check git status
```bash
git status
```

2. Add all changes
```bash
git add .
```

3. Commit with message
```bash
git commit -m "Update: [DESCRIBE_CHANGES]"
```

4. Push to current branch
```bash
git push
```

**Note**: You'll need to replace `[DESCRIBE_CHANGES]` with an actual commit message when running this workflow.
